نظرة عامة على Kali Linux

Kali Linux Overview

ما هي Kali Linux؟

Kali Linux هي توزيعة لينكس مبنية على Debian، ومصممة خصيصاً لاختبار الاختراق وأمن المعلومات.

لماذا Kali Linux؟

  • مفتوحة المصدر: مجانية بالكامل وقابلة للتعديل
  • 600+ أداة: مثبّتة مسبقاً وجاهزة للاستخدام
  • متخصصة: مصممة للهاكرز الأخلاقيين ومتقني الاختراق
  • محدثة: تُحدّث بشكل دوري
  • مجتمع كبير: دعم ومساعدة من آلاف المستخدمين

بنية Kali Linux

Kali Linux (Debian-based) ├── GNOME Desktop Environment (واجهة سطح المكتب) ├── Terminal (ال Sheldon - أهم أداة) ├── Pre-installed Tools (600+ أداة) ├── Custom Kernel (نظام مخصص) └── Rolling Release Model (تحديث مستمر)

سطح مكتب Kali

عندما تسجّل الدخول، ستظهر لك واجهة سطح المكتب. أهم شي هو Terminal (النافذة السوداء).

كيف تفتح Terminal

  • اضغط على أيقونة Terminal في شريط المهام
  • أو اضغط Ctrl + Alt + T
  • أو ابحث في Applications عن Terminal

ال Terminal هو الأداة الأساسية

في هذا الكورس، سنقضي معظم الوقت في الـ Terminal. كل الأوامر ستُكتب هنا.

معلومة مهمة

كثير من الهاكرز يفضلون Terminal لأنه أسرع وأقوى من الواجهة البصرية. تعلّم التعامل مع Terminal هو مفتاح النجاح في هذا المجال.

أوامر التحديث الأول

أول شي يجب فعله بعد التثبيت هو تحديث النظام:

# تحديث قائمة الحزم sudo apt update # ترقية كل الحزم المثبّتة sudo apt upgrade # تنظيف الحزم المؤقتة sudo apt autoremove

هذا يضمن أن لديك أحدث إصدارات الأدوات والنظام.

📝 تمرين 5: استكشاف Kali Linux
  1. افتح Terminal
  2. شغّل أمر sudo apt update && sudo apt upgrade -y
  3. افتح قائمة Applications وشاهد الأدوات المتاحة
  4. افتح أي أداة (مثل Nmap) وشاهد كيف تعمل
  5. عُد للTerminal واكتب أمر ls /usr/bin لرؤية الأوامر المتاحة
💡 ماذا سترى

عندما تكتب ls /usr/bin، سترى قائمة طويلة بالأوامر والأدوات المتاحة. هذا يُظهر لك مدى قوة Kali Linux!

جرّب أيضاً:
