Lesson 07

التنقل في نظام الملفات

Navigating the File System

بنية المجلدات في Linux

Linux يستخدم بنية شجرية (Tree Structure) لتنظيم الملفات والمجلدات. كل شي يبدأ من الجذر /.

/ ← الجذر (Root) ├── bin/ ← الأوامر الأساسية ├── boot/ ← ملفات الإقلاع ├── dev/ ← الأجهزة ├── etc/ ← إعدادات النظام ├── home/ ← مجلدات المستخدمين │ └── [username]/ │ ├── Desktop/ │ ├── Documents/ │ ├── Downloads/ │ └── .bashrc ├── lib/ ← المكتبات ├── media/ ← الأجهزة الخارجية ├── mnt/ ← نقاط التثبيت ├── opt/ ← برامج إضافية ├── proc/ ← معلومات العمليات ├── root/ ← مجلد المدير ├── sbin/ ← أوامر النظام ├── tmp/ ← ملفات مؤقتة ├── usr/ ← برامج المستخدمين └── var/ ← ملفات متغيرة

الأوامر الأساسية للتنقل

1. pwd - عرض المجلد الحالي

pwd اختصار لـ Print Working Directory. يعرض أين أنت الآن.

# عرض المجلد الحالي pwd # النتيجة: /home/kali

2. ls - عرض المحتويات

ls اختصار لـ List. يعرض كل الملفات والمجلدات في المجلد الحالي.

# عرض الملفات العادية ls # عرض التفاصيل (الصلاحيات، الحجم، التاريخ) ls -l # عرض كل شيء (بما في ذلك الملفات المخفية) ls -la # عرض الأحجام بشكل مقروء (KB, MB, GB) ls -lh # ترتيب حسب تاريخ التعديل ls -lt # ترتيب حسب الحجم ls -lS

3. cd - الانتقال بين المجلدات

cd اختصار لـ Change Directory. ينقلك لمجلد آخر.

# الانتقال لمجلد محدد cd /home/kali # الانتقال للمجلد الرئيسي (Home) cd ~ # أو cd # الرجوع للمجلد الأب cd .. # الرجوع للمجلد السابق cd - # الانتقال لمجلد Downloads cd Downloads # الانتقال للمجلد من مسار مطلق cd /var/log

4. mkdir - إنشاء مجلدات

# إنشاء مجلد واحد mkdir newfolder # إنشاء مجلدات متعددة mkdir dir1 dir2 dir3 # إنشاء مجلدات متداخلة mkdir -p parent/child/grandchild

5. touch - إنشاء ملفات فارغة

# إنشاء ملف فارغ touch newfile.txt # إنشاء عدة ملفات touch file1.txt file2.txt file3.txt # إنشاء ملفات بنمط touch file{1..10}.txt

شرح تفصيلي لـ ls -l

عندما تكتب ls -l، سترى something مثل:

-rwxr-xr-- 1 kali kali 4096 Jan 15 10:30 file.txt

هذا يعني:

- rwx r-x r-- 1 kali kali 4096 Jan 15 10:30 file.txt │ │││ │││ │││ │ │ │ │ │ │ │ │││ │││ │││ │ │ │ │ │ └── اسم الملف │ │││ │││ │││ │ │ │ │ └── تاريخ التعديل │ │││ │││ │││ │ │ │ └── الحجم (بايت) │ │││ │││ │││ │ │ └── اسم المستخدم (المالك) │ │││ │││ │││ │ └── عدد الروابط │ │││ │││ │││ └── نوع الملف │ │││ │││ └┴┴── صلاحيات الجميع (Others) │ ││┘ └┴┴────── صلاحيات المجموعة (Group) └┴┴────────── صلاحيات المالك (Owner)
نصيحة سريعة

اكتب alias ll='ls -la' في ملف .bashrc لتصنع أمر ll اختصاراً لـ ls -la.

📝 تمرين 7: تمارين التنقل
  1. اكتب pwd لعرض مجلدك الحالي
  2. اكتب cd / للذهاب للجذر
  3. اكتب ls لرؤية المجلدات
  4. اكتب cd home ثم cd مرة أخرى
  5. أنشئ مجلد باسم practice باستخدام mkdir
  6. اانتقال داخله usando cd
  7. أنشئ 3 ملفات فارغة باستخدام touch
  8. اكتب ls -la لرؤية كل شيء
