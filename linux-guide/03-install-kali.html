Lesson 03

تثبيت Kali Linux

Installing Kali Linux

تحميل Kali Linux

أول شي تحتاجه هو تحميل صورة Kali Linux (ISO file) من الموقع الرسمي.

خطوات التحميل

  1. ادخل على: kali.org/get-kali
  2. اختر "Kali Linux 64-bit (Installer Image)"
  3. اضغط "Download" وانتظر التحميل
  4. حجم الملف حوالي 3.8 GB
مهم

تأكد أنك تحمل النسخة الصحيحة. إذا جهازك 64-bit (معظم الأجهزة الحديثة)، حمّل النسخة 64-bit. إذا 32-bit، حمّل النسخة 32-bit.

إعداد الـ VM لتثبيت Kali

قبل تثبيت Kali، لازم تجهز جهاز افتراضي في VMware أو VirtualBox.

في VMware

  1. افتح VMware
  2. اضغط "Create a New Virtual Machine"
  3. اختر "Installer disc image file (ISO)" وحدد ملف Kali
  4. اختر "Linux" كنوع النظام و"Debian 10.x 64-bit" كإصدار
  5. أدخل اسم VM وحدد مكان الحفظ
  6. حدد حجم القرص الافتراضي (20GB كحد أدنى)
  7. اضغط "Finish"

في VirtualBox

  1. افتح VirtualBox
  2. اضغط "New"
  3. أدخل اسم (مثل: Kali Linux) ونوع Linux
  4. حدد حجم الذاكرة (4096 MB = 4GB كحد أدنى)
  5. اختر "Create a virtual hard disk now"
  6. اختر "VDI" ثم "Dynamically allocated"
  7. حدد الحجم (20GB كحد أدنى)

تثبيت Kali Linux

الآن جاهز لتثبيت Kali على الـ VM.

خطوات التثبيت

  1. شغّل الـ VM من ملف Kali ISO
  2. اختر "Graphical Install" من القائمة
نصيحة

"Graphical Install" أسهل للمبتدئين لأنه يعرض لك واجهة بصرية مع أزرار وخيارات واضحة.

  1. اختيار اللغة: اختر English أو Arabic حسب رغبتك
  2. اختيار المنطقة: حدد دولتك
  3. اختيار لوحة المفاتيح: اختر US أو حسب جهازك
  4. اسم المضيف (Hostname): اكتب اسماً (مثل: kali)
  5. اسم المستخدم: اكتب اسماً بسيطاً (مثل: hacker أو kali)
  6. كلمة المرور: ادخل كلمة مرور قوية (123456 مقبولة للممارسة)
  7. التقسيم (Partitioning): اختر "Guided - use entire disk"
  8. اختر "All files in one partition"
  9. اختر "Finish partitioning and write changes to disk"
  10. اختر "Yes" لكتابة التغييرات
  11. انتظر حتى ينتهي التثبيت (10-20 دقيقة)
  12. اضغط "Continue" لإعادة التشغيل

بعد التثبيت

بعد إعادة التشغيل:

  1. سجّل الدخول بكلمة المرور التي أدخلتها
  2. ستظهر لك سطح مكتب Kali Linux
  3. افتح Terminal (النافذة السوداء) وتكون جاهز!
📝 تمرين 3: تثبيت Kali Linux
  1. حمّل Kali Linux من الموقع الرسمي
  2. أنشئ جهاز افتراضي في برنامجك
  3. ثبّت Kali باستخدام الخطوات أعلاه
  4. سجّل الدخول وافتح Terminal
  5. اكتب أمر whoami وتأكد أنه يعرض اسم المستخدم
💡 التحقق من التثبيت

افتح Terminal واكتب:

whoami

يجب أن يعرض اسم المستخدم الذي أنشأته. مثلاً: kali أو hacker

أيضاً جرّب:

lsb_release -a

سيعرض معلومات Kali Linux والإصدار.
