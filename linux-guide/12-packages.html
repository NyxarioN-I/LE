تثبيت وتحديث الأدوات

Installing and Updating Tools

مدير الحزم apt

apt (Advanced Package Tool) هو مدير الحزم الرئيسي في Kali Linux وتبنيان Debian وUbuntu.

الأوامر الأساسية

# تحديث قائمة الحزم (يجب تشغيله أولاً) sudo apt update # ترقية كل الحزم المثبّتة sudo apt upgrade # تثبيت حزمة جديدة sudo apt install [اسم-الحزمة] # إزالة حزمة sudo apt remove [اسم-الحزمة] # إزالة حزمة مع إعداداتها sudo apt purge [اسم-الحزمة] # البحث عن حزمة apt search [كلمة] # عرض الحزم المثبّتة apt list --installed # تنظيف الحزم المؤقتة sudo apt autoremove

الفرق بين update و upgrade

apt update apt upgrade
يُحدّث قائمة الحزم المتاحة فقطيُحدّث الحزم المثبّتة فعلياً
لا يثبّت أي شييثبّت التحديثات الجديدة
سريع جداًقد يستغرق وقتاً

تحديث النظام في خطوة واحدة

# تحديث وترقية وتنظيف في سطر واحد sudo apt update && sudo apt upgrade -y && sudo apt autoremove -y
نصيحة

شغّل هذا الأمر كل يوم أو يومين للحفاظ على نظامك محدّثاً وآمناً.

تثبيت أدوات مهمة للهاكرز

أدوات الشبكة

# Nmap - ماسح الشبكات والمنافذ sudo apt install nmap # Netcat - أداة الشبكة sudo apt install netcat # Wireshark - محلل حركة الشبكة sudo apt install wireshark

أدوات الاختراق

# Nikto - ماسح ثغرات الويب sudo apt install nikto # Hydra - هجمات القوة الغاشمة sudo apt install hydra # SQLMap - هجمات SQL Injection sudo apt install sqlmap

أدوات كسر كلمات المرور

# John the Ripper sudo apt install john # Hashcat sudo apt install hashcat

أدوات الشبكات اللاسلكية

# Aircrack-ng sudo apt install aircrack-ng

أدوات أخرى مفيدة

# Git - إدارة الأكواد sudo apt install git # Python3 pip - مدير حزم Python sudo apt install python3-pip # Vim - محرر متقدم sudo apt install vim # HTOP - عرض العمليات sudo apt install htop

عرض الحزم المثبّتة

# عرض كل الحزم المثبّتة apt list --installed # البحث عن حزمة معينة apt search nmap # معرفة معلومات عن حزمة apt show nmap
📝 تمرين 12: تثبيت الأدوات
  1. شغّل sudo apt update لتحديث القائمة
  2. ثبّت Nmap بـ sudo apt install nmap
  3. تأكد من التثبيت بـ nmap --version
  4. ثبّت حزمة أخرى من اختيارك
  5. شغّل apt list --installed | grep [اسم-الحزمة] للتأكد
💡 الحل
sudo apt update sudo apt install nmap -y nmap --version # مثال آخر sudo apt install htop -y apt list --installed | grep htop
