البرمجة النصية Bash

Scripting with Bash

ما هي Bash Scripting؟

Bash (Bourne Again Shell) هي لغة برمجة نصية تُستخدم لأتمام المهام في Linux. تقدر تكتب سكربتات (ملفات تحتوي أوامر) لتنفيذ مهام معينة تلقائياً.

لماذا نتعلم Bash؟

  • أتمتة المهام المتكررة: بدل ما تكتب نفس الأمر 100 مرة
  • دمج عدة أوامر: تنفيذ مهام معقدة في ملف واحد
  • بناء أدوات مخصصة: أدوات اختراق مبنية حسب احتياجك
  • tiết kiệm الوقت: سكربت واحد يسوي شغل ساعات

الهيكل الأساسي للسكربت

#!/bin/bash # هذا سطر التعليق الأولي # يُخبر النظام أن هذا ملف Bash # كودك هنا echo "مرحباً بالعالم!"

خطوات تشغيل السكربت

# 1. إنشاء الملف touch myscript.sh # 2. تعديل الملف nano myscript.sh # 3. جعله قابل للتنفيذ chmod +x myscript.sh # 4. تشغيله ./myscript.sh

قراءة المدخلات

#!/bin/bash # قراءة نص من المستخدم echo "أدخل اسمك:" read name echo "مرحباً $name!" # قراءة بدون عرض النص المكتوب read -s -p "أدخل كلمة المرور: " password echo "" echo "تم القبول!"

المتغيرات

#!/bin/bash # تعريف متغير name="Kali" version=2024 pi=3.14 # طباعة متغير echo "نظام التشغيل: $name" echo "الإصدار: $version" # حسابات بسيطة a=10 b=20 sum=$((a + b)) echo "المجموع: $sum"

الشروط (if/else)

#!/bin/bash echo "أدخل رقم:" read number if [ $number -gt 10 ]; then echo "الرقم أكبر من 10" elif [ $number -eq 10 ]; then echo "الرقم يساوي 10" else echo "الرقم أقل من 10" fi

الحلقات (Loops)

1. For Loop

#!/bin/bash # طريقة 1: قائمة بسيطة for i in 1 2 3 4 5 do echo "الرقم: $i" done # طريقة 2: نطاق أرقام for i in {1..10} do echo "العدّاد: $i" done # طريقة 3: باستخدام seq for i in $(seq 1 10) do echo "العدّاد: $i" done # طريقة 4: تكرار على ملفات for file in *.txt do echo "ملف: $file" done

2. While Loop

#!/bin/bash count=1 while [ $count -le 5 ] do echo "العدّاد: $count" ((count++)) done

3. Until Loop

#!/bin/bash count=1 until [ $count -gt 5 ] do echo "العدّاد: $count" ((count++)) done

أمثلة عملية

مثال 1: ماسح بسيط

#!/bin/bash # Simple Network Scanner echo "==========================" echo " ماسح الشبكة البسيط" echo "==========================" echo "" echo -n "أدخل نطاق IP (مثال: 192.168.1): " read range echo "" echo "جاري المسح..." echo "" for i in $(seq 1 254) do ping -c 1 $range.$i | grep "64 bytes" & done echo "" echo "انتهى المسح!"

مثال 2: فحص المنافذ

#!/bin/bash # Simple Port Scanner echo "==========================" echo " فحص المنافذ البسيط" echo "==========================" echo "" echo -n "أدخل عنوان IP: " read ip for port in 21 22 23 25 53 80 110 143 443 445 993 995 3389 do (echo >/dev/tcp/$ip/$port) 2>/dev/null && echo "المنفذ $port مفتوح" done

مثال 3: أداة استطلاع

#!/bin/bash # Basic Recon Tool echo "==========================" echo " أداة الاستطلاع" echo "==========================" echo "" echo -n "أدخل الهدف: " read target echo "" echo "[*] جاري فحص الاتصال..." ping -c 2 $target echo "" echo "[*] جاري مسح المنافذ..." nmap -F $target echo "" echo "[*] انتهى!"
