Lesson 10

عرض وإنشاء وتعديل الملفات

Viewing, Creating, and Editing Files

عرض محتوى الملفات

1. cat - عرض كامل الملف

# عرض كامل الملف cat file.txt # عرض عدة ملفات cat file1.txt file2.txt

2. head - عرض أول أسطر

# عرض أول 10 أسطر (الافتراضي) head file.txt # عرض أول 20 سطر head -n 20 file.txt # عرض أول 5 أسطر head -5 file.txt

3. tail - عرض آخر أسطر

# عرض آخر 10 أسطر (الافتراضي) tail file.txt # عرض آخر 20 سطر tail -n 20 file.txt # متابعة الملف (مفيد للـ logs) tail -f /var/log/syslog

4. less و more - تصفح الملف

# تصفح الملف (مفاتيح: أسهم، q للخروج) less file.txt # تصفح الملف more file.txt

البحث في الملفات

grep - البحث عن نص محدد

# البحث عن كلمة في ملف grep "كلمة" file.txt # بحث بدون تمييز الأحرف grep -i "كلمة" file.txt # عرض الأسطر مع أرقامها grep -n "كلمة" file.txt # عرض عدد التكرارات grep -c "كلمة" file.txt # بحث في كل المجلدات grep -r "كلمة" /path/to/directory/ # بحث بكلمة كاملة فقط grep -w "كلمة" file.txt # عرض سطر قبل وبعد التطابق grep -B 2 -A 2 "كلمة" file.txt

إنشاء وتعديل الملفات

1. touch - إنشاء ملف فارغ

# إنشاء ملف فارغ touch newfile.txt # إنشاء عدة ملفات touch file{1..10}.txt

2. echo - إنشاء ملف بنص

# إنشاء ملف بنص (يمسح المحتوى القديم) echo "مرحباً بالعالم" > hello.txt # إضافة نص للملف (追加) echo "سطر إضافي" >> hello.txt # إنشاء ملف مع عدة أسطر echo -e "السطر الأول\nالسطر الثاني\nالسطر الثالث" > multi.txt

3.cp و mv - النسخ والنقل

# نسخ ملف cp file.txt backup.txt # نسخ مجلد cp -r folder/ backup_folder/ # نقل ملف mv file.txt /new/location/ # إعادة تسمية ملف mv oldname.txt newname.txt

4. rm - حذف ملفات

# حذف ملف rm file.txt # حذف مجلد وتحته rm -rf folder/ # حذف بدون تأكيد rm -f file.txt

المحررات

nano - المحرر السهل

محرر سهل جداً للمبتدئين.

# فتح ملف للتعديل nano file.txt # أوامر nano: # Ctrl + O → حفظ # Ctrl + X → خروج # Ctrl + K → قص سطر # Ctrl + U → لصق # Ctrl + W → بحث # Ctrl + C → عرض رقم السطر الحالي

vim - المحرر المتقدم

محرر قوي لكنه يحتاج تدريباً.

# فتح ملف vim file.txt # أوامر vim: # ESC → للخروج من وضع الكتابة # i → للدخول لوضع الكتابة (Insert) # :wq → حفظ وخروج # :q! → خروج بدون حفظ # dd → حذف سطر # yy → نسخ سطر # p → لصق # /text → بحث عن text

أوامر مفيدة أخرى

# عدّ عدد الأسطر wc -l file.txt # عدّ الكلمات wc -w file.txt # مقارنة ملفين diff file1.txt file2.txt # ترتيب محتوى ملف sort file.txt # إزالة الأسطر المكررة uniq file.txt
📝 تمرين 10: تمارين الملفات

  2. أضف 3 أسطر إضافية للملف
  3. اعرض أول سطرين من الملف
  4. ابحث عن كلمة معينة في الملف
  5. انسخ الملف لاسم جديد
  6. احذف الملف الأصلي
